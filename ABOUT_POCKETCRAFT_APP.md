# About PocketCraft (App)

PocketCraft is an Android app that lets you run and manage a real Minecraft Java Edition server from your phone.

## What It Is

PocketCraft turns your Android device into a lightweight Minecraft server host. Instead of renting a server or keeping a PC online, you can create and control your world directly on mobile.

## What You Can Do

- Create and run Minecraft Java server instances on Android
- Start, stop, and restart servers from a mobile dashboard
- Choose server versions and manage core server settings
- Install and manage plugins
- Monitor player activity and server status
- Configure advanced options for performance and gameplay
- Back up and restore worlds

## Why It Matters

PocketCraft is built for players and community admins who want an easier, lower-cost way to host Minecraft worlds:

- No desktop required for day-to-day management
- Faster setup for small private servers
- Portable server administration from anywhere
- Better control than shared public hosts for small groups

## Who It Is For

- Friends running private survival or mini-game servers
- Small communities that want simple self-hosting
- Creators testing plugins and server configs quickly
- Players who prefer mobile-first tools

## Core Idea

PocketCraft makes Minecraft server hosting feel like a normal mobile app experience: simple setup, clear controls, and practical admin tools without desktop-heavy workflows.
